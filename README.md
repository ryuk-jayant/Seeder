# Welcome to Seeder👋

This is an [Expo](https://expo.dev) project created for Andriod.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   expo start -c
   ```

You can start developing by editing the files inside the **app** directory. This project uses [file-based routing](https://docs.expo.dev/router/introduction).

## Get app on Andriod

1. Get ADB

   [visit](https://developer.android.com/tools/adb)

2. Connect Device
   connect real(developer mode) or Emulator via ADB check with
   ```bash
   adb devices
   ```
3. Start app
   ```bash
   npm run andriod
   ```
